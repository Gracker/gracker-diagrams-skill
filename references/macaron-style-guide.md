# 马卡龙信息图风格指南

## 一句话定义

**用暖色手绘信息图的单页摘要能力，把复杂知识体系压缩成一张可分享的竖屏长图。**

## 风格标签

- Hand-drawn educational infographic
- 马卡龙色系信息图
- 暖纸质感单页摘要
- Slides 单页视觉化
- 图形优先、文字点睛

## 美感硬约束

### 1. 背景
- 暖奶油纸底 `#F5F0E8`
- 不用纯白，不用深色
- 纸张质感，不是复杂纹理

### 2. 线条
- 轻微手绘抖动感（slight hand-drawn wobble）
- 不追求 Excalidraw 的粗黑描边
- 箭头用手绘波浪线（hand-drawn wavy arrows），不用正交箭头

### 3. 颜色
马卡龙色系圆角卡片，**色块不完全填满轮廓**（colors do not completely fill outlines）：
- 浅蓝 `#A8D8EA`
- 薄荷绿 `#B5E5CF`
- 薰衣草紫 `#D5C6E0`
- 浅桃 `#F4C7AB`
- 强调色：珊瑚红 `#E8655A`（关键词、重要数据、勾选标记）
- 线条与主文字：黑色
- 辅助标注：暖灰 `#6B6B6B`

### 4. 图标与装饰
- 简笔画卡通形象、示意图承载信息
- 涂鸦装饰点缀：小星星、下划线、小箭头
- 图形优先，文字仅用于标注和点睛
- 像好的 slides 一样——一眼看懂结构，细看理解细节

### 5. 字体
- 标题：粗体大号手绘字（bold, large, hand-drawn lettering），顶部居中
- 区域内：粗体关键词 + 暖灰小字短标签（2-5 词）
- 底部金句：粗体居中

### 6. 信息结构
根据内容自动选择最佳视觉布局：
- 流程 → 箭头串联
- 对比 → 左右分栏
- 循环 → 环形
- 组成 → 并列卡片
- 层级 → 嵌套

用圆角色块、气泡、虚线框等容器分区，区域间用手绘波浪箭头连接并标注简短关系词。

### 7. 留白
- 充足留白，干净构图
- 信息密度适中——比白板风密，但不是海报满版

### 8. 底部金句
- 图片底部一句粗体居中总结
- 概括核心观点
- 是信息图的收束点

## 默认比例

- **竖屏优先**：`1024x1536`（3:4）或 `768x1024`
- 横屏作为降级：`1536x1024`（仅在内容明确是横向结构时使用）
- 用户指定比例时听从用户

## 与默认风格的核心区别

| 维度 | 默认（白板架构图） | 马卡龙信息图 |
|------|-------------------|-------------|
| 背景 | 纯白 | 暖奶油纸 #F5F0E8 |
| 色块 | 外框/glow，不填满 | 马卡龙圆角卡片，不完全填满轮廓 |
| 箭头 | 正交 | 手绘波浪线 |
| 图标 | 黑白简化大图标 | 简笔画卡通 + 涂鸦装饰 |
| 标题 | 弱化/可省 | 粗体大号手绘字居中 |
| 密度 | 低密度白板感 | 中密度 slides 单页摘要 |
| 比例 | 横屏 16:9 | 竖屏 3:4 |
| 底部 | 无 | 金句收束 |
| 适用 | 架构关系、系统拆解 | 知识总结、方法论、Skill 概览、框架图 |

## 禁止项

- 纯白底（必须用奶油纸）
- 正交箭头（必须波浪线）
- 写实插画
- 萌系卡通
- 高饱和 neon
- corporate PPT 风格
- 满版无留白
- 无底部金句
