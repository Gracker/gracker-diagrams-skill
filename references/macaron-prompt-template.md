# 马卡龙信息图 Prompt 模板

把下面内容填入，生成最终 prompt：

```
你是一位擅长手绘风信息图的视觉设计师。请根据以下内容创作一张单页信息图。

风格要求

整体风格：Hand-drawn educational infographic on warm cream paper texture (#F5F0E8)。所有线条和形状带轻微手绘抖动感（slight hand-drawn wobble），整体干净清晰，像高质量演示文稿的单页视觉摘要。无写实元素。

配色方案：
- 信息区分区：用浅蓝 #A8D8EA、薄荷绿 #B5E5CF、薰衣草紫 #D5C6E0、浅桃 #F4C7AB 作为圆角卡片背景色。卡片边框用干净的直线，不要手绘抖动感——卡片框要工整利落。不要画气泡框、虚线框。
- 强调色：珊瑚红 #E8655A，用于关键词、重要数据、勾选标记等需要视觉突出的元素
- 线条与主文字：黑色
- 辅助标注：暖灰 #6B6B6B，字号较小

图形与文字平衡：文字是信息的绝对主体。尽量少用图标和装饰性插画——绝大多数信息图里的小图标不承载实际信息，反而分散注意力。只在以下情况使用图标：模块入口需要一个视觉锚点、或者某个概念确实无法用文字简洁表达。整体原则是「文字密度优先，图标克制使用」。像高质量技术博客的配图一样，远看结构清晰，近看文字有干货。

信息结构：根据内容自动选择最佳视觉布局（流程→箭头串联，对比→左右分栏，循环→环形，组成→并列卡片，层级→嵌套等）。区块间用手绘波浪箭头（hand-drawn wavy arrows）连接并标注简短关系词。背景色块做分区即可，不要画太多独立卡片框、虚线框、气泡——避免条条框框堆砌感，区块之间自然过渡。

文字层次：标题顶部居中，粗体大号手绘字（bold, large, hand-drawn lettering）；区域内粗体关键词 + 正常文字说明混排，允许完整句子和短语，暖灰小字用于辅助标注。

渲染细节：涂鸦装饰点缀（小星星、下划线、小箭头等），充足留白，干净构图。背景色用奶油纸底，区块间用不同浅色背景自然区分。

底部金句：图片底部一句粗体居中总结，概括核心观点。

Text labels (in Chinese):
<TEXT_LABELS>

Generate the infographic based on the content below:

<CONTENT>
```

## TEXT_LABELS 片段

格式：
```
- 标题: <title>
- 主模块: <module names comma-separated>
- 标注: <annotation texts>
- 底部金句: <one-sentence summary>
```
